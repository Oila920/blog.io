利用GitHub搭建一个独立的个人主页

September 14, 2017
  • for Oila

在开始前 ,我先说一下自己搭建这个博客的感受

        作为攻城狮 ,很早就想要一个属于自己主页或者博客 ,用于展示自己所学的知识 ,分享自己的开发经验.之前没有自己去搭建的时候 ,总是觉得那应该是一件不简单的事情 ,直到自己着手去折腾了 ,发现其实并不难 !

首先 ,介绍一下大致的步骤

1.购买域名以及godaddy账号申请

2.GitHub账号申请以及git库的配置

3.dnspod账号申请以及相关配置

4.重新设置godaddy的DNS地址指向dnspod

5.GitHub仓库管理软件的使用

那么 ,正式开始了 !

1.购买域名

进入这个网站GoDaddy(世界知名互联网域名注册商 ,请大家放心食用)

如果进到这个页面是英文的 ,放心 ,这个网站目前已经支持中文了 !点击左上角箭头所指的地方 ,切换一下就好了 ,推荐切换到香港区

接下来直接在输入框中输入自己想要的域名 ,比如输入wxq ,搜索

在搜索结果页面选择自己喜欢的域名 (不同结尾的域名有不同的功能 ,不然也不会买那么贵了T-T .做博客的话 ,选择一个便宜的就好了),点击加入购物车 ,成功后 ,箭头所指的"前往购物车"按钮就亮了 ,点击它

接下来的这个页面 ,都选择"谢谢 ,不必了!" ,还有一个"免费建立您的网站"的选项也不用勾选 ,这些东西我们都可以通过GitHub跟其他一些免费的途径进行实现 ,所以可以不用他家的

那么 ,终于来到了付款页面了 !但是不是都还木有账号怎么买啊 ?那就注册一个呗 ,注册完再回到购物车继续购买 ,据说网上有优惠码 ,上网搜GoDaddy优惠码试试，一般优惠幅度是 20%~ 30% 不等

注意:买完域名之后一定要记得去自己的邮箱查看激活邮件，否则域名激活不了

购买域名完成之后 ,我们先把它放一边 ,接着下一步

2.GitHub账号申请以及git仓库的配置

去我们熟悉的GitGub,如果有账户就直接登录 ,木有的话就注册一个

创建git仓库

只要填写仓库一下名字 ,就可以创建成功啦 !这个名字可以随意填写 ,可以把它看成是一个文件夹

接下来进到这个仓库的目录下 ,按图示创建一个必要的CNAME的文件和一个简单的html页面(其他的等后期再添加啦) ,创建好后记得点击页面最下面绿绿的按钮"Commit new file"提交一下更改

CNAME文件中填写我们刚才申请的域名(此步骤就是将我们的域名链接到github仓库 ,不过现在还是没办法使用 ,还需后续步骤)

接下来去激活一下GitHub Page

GitHub Pages 是什么？GitHub Pages 本用于介绍托管在 GitHub 的项目， 不过，由于他的空间免费稳定，用来做搭建一个博客再好不过了。这是官方对Github Page的介绍:点我跳转

在博客的仓库界面找到"Settings" ,点击进入本仓库设置

往下拉找到这个选项 ,选择"master branch" ,然后点击旁边的save

刷新页面 ,往下拉移到刚才位置 ,发现上面多了一个这样的链接 ,这就是我们git hub的博客!

但现在并不是我们想要的那个个性的域名 ,所以接下来继续做一些配置 ,打开终端(win是命令行) ,ping我们仓库IP ,格式是:ping https://github/你的github账户名.github.io ,比如我ping的就是ping https://github/oila920.github.io ,记录下ping出的IP地址 ,这个IP可能一段时间后会变化 ,以后需要更新

那么github这边的配置已经基本完成了 ,接下来去做下一步

3.dnspod账号申请以及相关配置

进入这个网站DNSPOD(国内最火的第三方免费DNS服务 ,请大家放心食用!) ,木有的账户的话就注册一个然后登陆进去来到这个页面 ,点击"域名解析",接着点击"添加域名",输入在godaddy购买的域名 ,然后确定

接着点击点击域名来到域名配置页面 ,记录类型为A的 ,在记录值那边填写刚才利用终端(win是命令行)ping出的IP地址;记录值为CNAME的 ,再记录值那边填写我们的仓库名 ,格式如图(github用名+.+上之前创建的仓库名+. 例如:Oila920.bolg.io.) ;还有两个记录类型为NS的 ,记录下来

4.重新设置godaddy的DNS地址指向dnspod

去到godaddy >我的产品页面 ,点击"DNS" ,

界面往下拉 ,找到网域名称伺服器 ,点击"变更" > "自定" ,填入刚才在DNSPOD记录下来的NS记录值f1g1ns1.dnspod.net和f1g1ns2.dnspod.net然后保存

致辞 ,已经完成了整个博客的搭建 ,在网址栏输入我们注册的域名 ,可以直接打开我们的博客啦!

5.GitHub仓库管理软件的使用

为了更好的管理我们的博客 ,让本地项目可以直接同步到git仓库 ,所以最后还有一个步骤

进到github仓库页面 ,点击"Open in desktop" ,此时如果已安装GitHub Desktop ,则会自动打开GitHub Desktop软件 ,并且弹出选择文件路径的弹窗 ,如果未安装则会自动跳转到下载GitHub Desktop的页面 ,下载安装后 ,打开GitHub Desktop登录GitHub账号

再去到github仓库页面 ,再重复以上步骤 ,点击"Open in desktop" ,此时就可以选择本地路径 ,将代码Pull到本地 ,接下来就可以在本地对项目进行修改了


致此 ,整个教程已经全部结束 ,如果有什么问题 ,大家可以联系我交流讨论!
